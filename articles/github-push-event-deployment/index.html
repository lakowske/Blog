Github push event deployment

Posted by: Seth Lakowske

Published: 2014-12-30

You can deploy code by simply pushing it to a Github and using github-webhook-deployer. When a Github webhook push event occurs, github-webhook-deployer pulls code and kills the current process Because the service was started using a daemon manager, such as forever, it will be restarted after it is killed and we are alive with our new code.
